## Instalar

Clonar repositório:
```bash
git clone https://github.com/RodrigoESTGL/Calculadora_SD.git && cd Calculadora_SD
```

Instalar dependências:
```bash
npm install
```

Correr o servidor em modo desenvolvimento:
```bash
nodemon /JS/server.js
```
